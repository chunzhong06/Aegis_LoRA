# Aegis-LoRA

**大语言模型 (LLM) 后门检测与靶向免疫防御终端**

Aegis-LoRA 提供了一套从“静态谱特征探测”到“物理级神经元切除”的端到端自动化安全工作流。通过现代化的 Web 控制台，帮助开发者一键净化被恶意毒化的 LoRA 适配器，保障大模型底层对齐安全。

## 核心特性

- **双擎免疫干预引擎**
  - **极速查杀 (Fast Cleanse):** 加载预计算的多任务域签名图谱，实现秒级的后门特征精准切除。
  - **深度重构 (Deep Cleanse):** 联机动态提取变体差分矩阵，强力剥离复杂的深层结构化后门（如代码注入、情感操纵等）。
- **静态无损扫描**
  - 利用轻量级统计学探测器捕捉权重张量的异常拓扑分布，无需全量加载基座模型即可完成初筛拦截。
- **极致显存隔离机制**
  - 采用 OS 级物理进程隔离与严格的 C++ 闭包张量回收策略，彻底根除高并发多任务调度下的 PCIe 显存颠簸与框架泄漏。
- **专业级离线审计**
  - 自动生成防篡改的 HTML/JSON 审计报告，直观可视化展现参数切除（Excised）与保留（Retained）的 L2 范数变化。

## 环境要求

建议运行环境：Windows / Linux 操作系统，并配备 NVIDIA GPU。

```bash
# 1. 获取项目代码
git clone https://github.com/chunzhong06/Aegis_LoRA.git
cd Aegis_LoRA

# 2. 安装依赖
pip install -r requirements.txt
```

## 快速启动

在项目根目录下执行主控脚本：

```bash
python main.py
```

_服务启动后，在浏览器中打开终端输出的本地地址（默认 `http://127.0.0.1:7860`）即可进入 Aegis-LoRA 可视化控制中心。_

## 操作指南

1. **挂载节点**: 在侧边栏输入基座模型与待检 LoRA 的物理路径。
2. **选择策略**: 根据安全需求，配置应用「极速免疫」或「深度免疫」。
3. **一键净化**: 系统将自动接管静态扫描、多域特征提取、参数切除及自愈微调全流程。
4. **安全上线**: 净化完毕后，模型将自动挂载至安全内存区，即可在主视窗中进行无害化推理测试。
